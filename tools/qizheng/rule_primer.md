# 七政四余基础规则

## 七政
- 太阳：主贵、父、目、君主之德。
- 太阴：主母、妻、财、情绪、田宅。
- 金星：主财帛、艺术、交际、婚姻。
- 木星：主福德、学问、官禄、贵人。
- 水星：主智谋、口舌、文书、迁移。
- 火星：主武职、疾厄、冲突、魄力。
- 土星：主田宅、劳碌、限制、迟缓。

## 四余
- 紫气：木之余，主清高、福德、隐逸。
- 月孛：水之余，主情欲、偏才、暗昧。
- 罗睺：火之余，主灾祸、执着、变异。
- 计都：土之余，主暗耗、业力、秘密。

## 十二宫
命宫、兄弟、夫妻、子女、财帛、疾厄、迁移、奴仆、官禄、田宅、福德、父母。

## 推断要点
1. 先看命宫主星与身宫主星。
2. 再看日、月是否得时得地。
3. 七政四余在十二宫的分布决定各领域吉凶。
4. 四余犯主星多为波折，七政相生多为顺遂。
