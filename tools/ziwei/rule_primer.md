# 紫微斗数规则入门

## 1. 命盘结构

紫微斗数以农历出生年月日时排出十二宫：命宫、兄弟、夫妻、子女、财帛、疾厄、迁移、仆役、官禄、田宅、福德、父母。命宫与身宫为论断核心。

## 2. 主星与辅星

主星十四颗：紫微、天机、太阳、武曲、天同、廉贞、天府、太阴、贪狼、巨门、天相、天梁、七杀、破军。辅星如左辅、右弼、文昌、文曲、天魁、天钺、禄存、天马、擎羊、陀罗、火星、铃星、地空、地劫等。

## 3. 四化

四化（禄、权、科、忌）是紫微斗数的重要动线。化禄主顺遂与财禄，化权主掌控与变动，化科主名声与解厄，化忌主执着与波折。

## 4. 宫位吉凶判断

- 命宫主一生格局、性情与运势基调。
- 身宫主后天努力方向与中晚年重心。
- 事业宫看官禄、事业方向与贵人助力。
- 财帛宫看求财方式与稳定度。
- 夫妻宫看婚姻质量与配偶特质。
- 疾厄宫看体质强弱与易患部位。
- 迁移宫看外出、社交与环境适应。

## 5. 常用格局

- 紫微天府：稳重有领导力，宜管理行政。
- 七杀破军：变动大，宜武职、技术、创业。
- 日月并明：光明磊落，利名声与公职。
- 巨日同宫：口才佳，利销售、传播、法律。

## 6. 评分原则

综合主星组合、辅星吉凶、四化流向、煞星冲破程度给出 0-100 评分。高分需主星庙旺、辅星吉多、四化流通；低分多见煞忌冲破、主星落陷。
