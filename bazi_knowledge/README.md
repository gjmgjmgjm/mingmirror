## 使用方法

### 按八字检索
直接搜索八字，如：
```
甲申 癸酉 壬子 甲辰
```

### 按术语检索
搜索命理术语，如：
```
伤官
正财
身强
格局
```

### 按结论检索
搜索结论性关键词，如：
```
好命
财运好
婚姻好
```

## 知识库特点

1. **结构化**：每个案例都有统一的格式
2. **可检索**：支持多种检索方式
3. **可学习**：包含命理师分析和命主反馈
4. **可扩展**：可以添加更多案例
